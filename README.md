# features
``custom reading rate``,
``5 rereads for questions``,
``filters the lowest word count story``

# commands
### replace {} with ur value

changinng the reading rate (reload req)
``rate="{amount}"``,

clearning the log
``cls=""``,

getting the lowest wordcount story
``lowest=""``

getting vocab answer
``contex=""``
